# 井冈山的斗争 -- 思维提炼报告

---

## 模块一：核心算法一句话总结

**先找能活下来的根据地，再谈怎么打赢。活下去的本质是五个要素的闭环：有人、有组织、有武装、有地利、有钱粮。少一个，迟早崩盘。**

---

## 模块二：历史困境与破局逻辑

### 困境：一群被逼到山里的散兵，凭什么活下去？

当时的局面可以用四个字概括：**弹尽粮绝**。没有正规军编制，没有后方补给线，被各方势力围追堵截，内部思想混乱，随时可能散伙。

换到今天的语境：你带着几个合伙人，没什么资金，没有背景，产品还不成熟，市场巨头环伺，团队随时可能有人提离职。怎么搞？

破局逻辑分三层：

**第一层：活下来比什么都重要。** 不是想着怎么打赢全国市场，而是先找一个敌人管不到、自己能吃饭的地方扎下根。罗霄山脉中段就是这个"根据地"——进可攻退可守，群众基础还行，两边省的缝隙谁都不想管。说白了就是：先找一个竞争没那么激烈的细分市场活下来。

**第二层：根据地不是白捡的，要靠五个硬条件撑住。** 群众基础（用户认可）、党组织（团队执行力）、红军（产品/业务能力）、地形（市场定位）、经济力（现金流）。这五根柱子缺一根，整个模型就塌。这不是什么战略规划，这是生存检查清单。

**第三层：活下来之后，怎么发展是有节奏的。** 不是一味猛冲，也不是一直苟着。要根据外部环境的变化，动态调整进攻和防守的比例。这就是后面要展开的核心算法。

## 模块三：顶层思维模型提取

### 模型一：环境周期判断法 -- 你现在处于"破裂期"还是"稳定期"？

这是整篇文章最核心的思维模型。

**核心逻辑：外部环境决定了你该用什么策略，而不是你的主观意愿。**

- **破裂期（市场动荡、对手内讧、行业洗牌）：** 可以冒进。因为对手顾不过来，你出手的容错率高。但即使冒进，也要同时把核心阵地夯实——不是光顾着抢地盘，根基不能丢。
- **稳定期（市场格局固化、巨头防守严密）：** 必须稳扎稳打，最忌分兵冒进。

**反面教训：** 八月失败的根本原因就是在稳定期用了冒进策略。分兵出击湘南，结果主力被各个击破，根据地丢了大半。团队的兵都没心思打仗了，想回家，加上天气热、补给断、和后方失联——这不是天灾，这是战略判断错误。

**商业翻译：** 很多创业公司死掉，不是因为方向错了，而是在行业格局已经稳定的时候，还按"跑马圈地"的打法去烧钱扩张。烧完钱发现根基本来就没打牢。

### 模型二：集中兵力原则 -- 永远不要分兵

**原文原话："集中红军相机迎击，反对分兵。"**

这个原则极其朴素但极其重要：**在资源有限的情况下，分散兵力等于自杀。**

为什么？因为分兵之后，每一股力量都不够强，打不过对手的任何一支主力。而且分兵带来致命的管理成本——指挥链断裂、信息不同步、互相救不了。

**八月失败的再验证：** 分兵向湘南，导致边界兵力空虚，敌人趁机反扑，根据地一夜回到解放前。

**商业翻译：**
- 资源有限的时候，集中所有力量打一个点，把这个点打穿。
- 不要同时做三个产品、拓展五个城市、铺开十条业务线。你以为是多元化，其实是慢性自杀。
- "单点突破"不是保守，是资源约束下的最优解。

### 模型三：波浪式推进 -- 不要一口吃成胖子

正确的打法不是一窝蜂冲出去，而是**像波浪一样，往前推一段，巩固一段，再推下一段。**

为什么要这样？因为每推一段，你需要：
- 把当地群众发动起来（获取用户）
- 建立地方党组织（搭建本地运营团队）
- 发展地方武装（培养本地护城河）
- 搞好经济供给（建立自我造血能力）

如果只顾往前冲，后面就是烂摊子。白区（新区）的群众还没起来，你的军队一走，敌人回来，一切归零。

**商业翻译：** 每进入一个新市场，必须先做到"可自运转"再拓展下一个。很多连锁品牌开了一堆店，但每家店都是亏损的，总部一断血，全盘崩盘。这就是典型的只推波、不巩固。

### 模型四：组织建设的底层逻辑 -- 支部建在连上

**原文原话："红军像一个火炉，俘虏兵过来马上就熔化了。"**

这句话揭示了一个极其深刻的组织管理洞察：**一个组织的凝聚力不是靠口号，而是靠制度设计。**

具体怎么做？

1. **待遇平等：** 军长到伙夫，一律五分钱伙食。这不是"大锅饭"思维，而是**消除内部层级带来的怨恨和对立**。当所有人吃一样的饭，士兵就不会觉得"当官的在拿我当炮灰"。
2. **经济透明：** 士兵管伙食，还有"伙食尾子"（结余归士兵）。**把钱的事透明化，是消除内部猜疑的最直接手段。**
3. **组织下沉：** 支部建在连上。不是只有高层有党组织，而是**每一个最小的战斗单元都有自己的组织核心**。这意味着即使连队被切断联系，不会溃散，因为每个连队都有自己能做主的人。
4. **思想工作：** 不是靠恐吓和惩罚维持纪律，而是靠说服和教育。俘虏兵过来为什么能"马上熔化"？因为他们感受到了跟之前完全不同的组织氛围。

**商业翻译：**
- 创业公司最怕什么？核心骨干一走，整个业务线瘫痪。为什么？因为组织能力全挂在几个人身上，没有沉淀到组织层面。
- "支部建在连上"翻译过来就是：**每个最小业务单元都要有独立运转的能力，不依赖任何单一英雄。**
- 待遇平等不是说老板和员工工资一样，而是说**分配规则必须透明、公平，且核心团队要有"与一线同甘"的姿态。**

### 模型五：中间阶级博弈论 -- 别把能争取的人推到对面

**原文洞察：打击过重导致中间阶级反水，白色恐怖一到，他们马上投降豪绅。**

这是一个极其精妙的博弈论洞察：**中间阶级（富农、小地主）不是你的敌人，但如果你把他们当敌人打，他们就会变成你的敌人。而且他们会带着资源和人脉投奔你的对手，让你的处境更加恶化。**

经济封锁一来，中间阶级第一个受不了。他们本来可以帮你打破封锁（因为他们有钱、有渠道），但你把他们的利益打光了，他们凭什么帮你？

**商业翻译：**
- 在竞争中，永远要分清谁是敌人、谁是朋友、谁是中间派。
- 中间派（供应商、渠道商、中立竞品、潜在合作方）的态度决定了你的生存空间。
- 对待中间派的策略不是"彻底消灭"，而是**争取、分化、利用**。把他们推到对面，你的敌人就多了一个有资源有能力的帮手。
- 很多公司犯的错误：对合作伙伴压榨太狠，结果对方转头和竞品合作。这不是对方"不讲义气"，是你自己把人逼走的。

### 模型六：根据地五要素模型 -- 生存的完整检查框架

五个条件缺一不可：

| 要素 | 商业翻译 | 自查问题 |
|------|---------|---------|
| 很好的群众 | 有认可你的目标用户 | 你的用户真的爱你吗？还是只是贪便宜？ |
| 很好的党 | 有执行力的团队 | 你的团队能打硬仗吗？还是只会PPT？ |
| 相当力量的红军 | 有竞争力的产品/业务能力 | 你的产品在市场上能打吗？ |
| 便于作战的地势 | 有利的市场定位 | 你选的赛道好打吗？巨头看不上还是打不过？ |
| 足够给养的经济力 | 有可持续的现金流 | 你的钱能撑多久？造血能力在哪？ |

**关键洞察：这五个条件是相互强化的。** 群众基础好 → 兵源和情报就足 → 军队就强 → 根据地就稳固 → 经济就有保障 → 群众就更信任你。这是一个正向飞轮。反之，任何一个环节断了，飞轮就停转。

## 模块四：现代场景映射

### 场景一：创业公司从0到1

**你的井冈山在哪里？**

不要一上来就想做全国市场。先找一个巨头看不上、你够得着、需求真实存在的细分领域，扎下去。这个领域就是你的"罗霄山脉中段"。

然后做五要素自检：
- 这个领域的用户真的需要你吗？（群众基础）
- 你的团队有能力深耕这个领域吗？（党组织）
- 你有拿得出手的产品吗？（红军）
- 这个领域的竞争环境对你有利吗？（地形）
- 你有足够活过18个月的资金吗？（经济力）

如果五个条件都满足，就可以开始"波浪式推进"——先把一个城市或一个细分人群做透，再做下一个。

### 场景二：职场中的"根据地"建设

**在公司里，你的"根据地"是什么？**

是你在公司里不可替代的那块能力。不是什么都会一点，而是在某个关键领域做到团队第一甚至公司最强。

- **群众基础** = 和你协作的人都说你靠谱
- **党组织** = 你在关键决策链上有影响力
- **红军** = 你的核心技能真的硬
- **地形** = 你所在的部门或业务线是公司重点
- **经济力** = 你的薪资和职业发展空间有保障

### 场景三：副业/个人品牌

**不要同时做五个副业，集中兵力打一个。**

选一个方向（你的井冈山），把内容、渠道、产品全部聚焦在这一个方向上。先做到能稳定变现（经济力），再考虑拓展。

很多做副业的人最大的问题：今天做短视频，明天写公众号，后天搞知识星球。这就是"分兵冒进"，结果每个都做不好。

## 模块五：行动指导 SOP（实战检查清单）

### 一、环境判断检查（每季度做一次）

- [ ] 当前行业处于"破裂期"还是"稳定期"？（判断标准：主要竞品是否在收缩、融资环境是否在恶化、政策是否有重大变动）
- [ ] 如果是破裂期：列出3个可以冒进抢占的机会窗口，同时确认核心根据地是否稳固
- [ ] 如果是稳定期：暂停一切扩张计划，回到核心业务做深度优化
- [ ] 你最近一次的战略调整，是基于环境变化做出的，还是拍脑袋决定的？

### 二、资源集中度检查（每月做一次）

- [ ] 你当前的主要精力（70%以上）是否集中在一个核心方向上？
- [ ] 列出你正在做的所有项目/业务线，砍掉不能产生80%收益的20%项目之外的所有项目
- [ ] 你的团队是否每个人都清楚"当前最重要的那件事"是什么？
- [ ] 有没有在做"撒胡椒面"式的资源分配？（每个项目都投一点，每个都做不好）

### 三、推进节奏检查（每次制定计划时）

- [ ] 你的扩张计划是否符合"波浪式推进"？即：每推进一步，是否已经确保前一步已经稳固？
- [ ] 新开拓的市场/业务，是否已经建立了"可自运转"的本地团队？
- [ ] 如果主力被抽走，已占领的市场能否自保？（如果不能，就不要往前推）

### 四、组织健康度检查（每半年做一次）

- [ ] 你的最小业务单元（相当于"连"）是否有独立运转的能力？还是全靠你一个人？
- [ ] 核心团队的利益分配是否透明、公平？有没有因为分配不公导致的暗流涌动？
- [ ] 团队中是否有"俘虏兵变铁军"的案例？（即新人快速融入并产生价值的能力）
- [ ] 你是否在每个关键节点都设置了"支部"（即有明确负责人和决策权的基层组织）？

### 五、生态关系检查（每季度做一次）

- [ ] 列出你的合作伙伴/供应商/渠道商，对每个人打标签：坚定支持者 / 中间派 / 潜在对手
- [ ] 对"中间派"是否采取了争取策略？还是把他们当对手压榨？
- [ ] 最近有没有因为利益分配问题，把原本可以合作的人推到了对手那边？
- [ ] 你是否建立了足够多的"中间阶级"缓冲带？（即在激烈竞争中保持一定数量的中立或友好关系）

### 六、五要素存活检查（创业/转型前必做）

- [ ] **群众**：目标用户是否真的有痛点？是否已经有用户自发传播？
- [ ] **组织**：核心团队是否磨合到位？是否有明确的分工和决策机制？
- [ ] **能力**：你的产品/服务是否有明确的核心竞争力？不是"我觉得好"，而是市场验证过。
- [ ] **定位**：你选的赛道是否足够聚焦？巨头是否真的短期内打不过来？
- [ ] **粮草**：你的资金是否够支撑到盈亏平衡？最坏的现金流预期是什么？

**如果以上六个维度中有两个以上亮红灯，不要启动，不要扩张，先回去修基本功。**
