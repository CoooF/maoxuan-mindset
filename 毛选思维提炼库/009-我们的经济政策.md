# 009-我们的经济政策（1934年1月）

## 模块一：核心算法一句话

**在资源极度受限的生死存亡期，先用利益绑定激活生产端，再用"自营+合作+外包"三层业务结构稳住供给底盘，最后通过做大蛋糕而不是杀鸡取卵来养活战争机器。**

一句话翻译：**别想着靠克扣供应商和加税活命，把蛋糕做大才是唯一的续命逻辑。**

---

## 模块二：历史困境与破局逻辑（20%）

### 1934年的死局

1934年初，苏区被国民党经济封锁围得铁桶一般。食盐进不来，钨砂出不去，连火柴肥皂都是奢侈品。根据地内部农民刚分到土地还在观望，工业基础约等于零，财政连打仗都养不起。换算成商业语言：**一家创业公司在被行业巨头全面封锁、供应链断裂、账上快没现金的情况下，怎么活？**

### 破局三步棋

**第一步：土地分配 = 股权激励。** 把土地分给农民，确认产权（确定地权），让每个人都知道"这是我的"。农民不是被动员的，是自己拼命干——因为劳动成果归自己。生产恢复并超过革命前水平，这不是奇迹，是利益绑定的力量。

**第二步：重新定义"经济"的优先级。** 不是搞什么重工业大项目，而是死磕农业——粮食、衣服、砂糖、纸张这些生存级物资。先活下来，再谈发展。翻译成商业语言：**创业公司的第一要务不是做平台做生态，是先把能造血的核心业务跑通。**

**第三步：不堵死私人的路，反而鼓励。** "只要不出于法律范围，不但不加阻止，而且加以提倡和奖励。"这句话放在1934年简直是认知炸弹——搞革命的人居然在鼓励私人经济。但逻辑很清楚：**资源不够的时候，必须让所有能干活的人都来干活，而不是只靠体制内那点力量。**

## 模块三：顶层思维模型提取（40%核心）

### 模型一：三层业务架构设计——自营/合作/外包的动态平衡

这是整篇文章最硬核的思维模型。

| 三层结构 | 1934年的原文 | 今天的商业翻译 | 战略定位 |
|---------|------------|--------------|---------|
| 国营经济 | 国营事业 | 自营核心业务 | 控制命脉，定规则 |
| 合作社经济 | 合作社事业 | 合资/联营/平台生态 | 放大杠杆，扩大覆盖 |
| 私人经济 | 私人事业 | 外包/供应商/自由市场 | 激活增量，分担成本 |

**核心逻辑：不是消灭私人经济，而是让自营+合作经过长期发展，自然地、逐渐地对私人经济取得领导地位。** 翻译：不是干掉所有外包商自己全干，而是让核心业务和合作生态长到足够大，自然形成对供应链的控制力。这是一套**渐进式控股战略**——不搞休克疗法，靠时间换空间。

**关键洞察：短期能容下的，长期才能领导。容不下就只剩下对抗，对抗消耗的是自己。**

### 模型二：利益绑定优先于思想动员

分土地 → 确地权 → 农民自己拼命 → 产量超革命前。

这套逻辑链条揭示了一个反常识的真相：**人不会因为你的口号拼命，只会因为利益归自己而拼命。** 所有的"激发劳动热情"，底层都是产权清晰+收益归己。

商业映射：
- 股权激励的本质不是"分钱"，是让核心人才产生"这是我的公司"的幻觉
- 阿米巴经营的本质不是"管理创新"，是把大公司的部门变成"自己的生意"
- 门店承包制的本质不是"制度变革"，是把员工变成有产者

**不解决"这个东西是谁的"这个问题，所有的KPI和OKR都是在推绳子。**

### 模型三：造血型财政 vs 抽血型财政

"从发展国民经济来增加财政收入，而不是靠增税。"

这句话可以直接挂在每个创业公司CEO的办公室墙上。

- **抽血型财政：** 预算不够？加税！供应商利润太高？压价！员工成本太高？裁员！短期账面好看，长期生产端萎缩。
- **造血型财政：** 预算不够？把业务做大！把营收做大！做大蛋糕之后财政收入自然增加。

**财政问题的本质不是分配问题，是生产问题。你分的是一个固定蛋糕还是一个正在长大的蛋糕，决定了所有 downstream 的决策质量。**

### 模型四：资源约束下的优先级排序

四个经济建设中心：农业→工业→外贸→合作社。不是并列的，是有严格的先后顺序。

- **农业第一：** 吃饭问题不解决，其他都是空中楼阁。对应商业：**核心造血业务第一，先活下来。**
- **工业第二：** 先自给再出口。对应商业：**先解决内部效率问题（自给），再考虑对外扩张（出口）。**
- **外贸第三：** 打通外部资源通道。对应商业：**建立供应链和销售渠道。**
- **合作社第四：** 组织化放大。对应商业：**平台化和生态化是最后阶段，不是起手式。**

**在资源受限时，排错优先级不是浪费资源的问题，是死活的问题。**

### 模型五：货币与实体的关系

"货币发行应根据国民经济发展需要，单纯财政需要只能放次要地位。"

翻译成商业语言：**融资节奏必须匹配业务增长节奏。** 为了补窟窿而融资（单纯财政需要），和为了扩张业务而融资（经济发展需要），结果天差地别。前者是拆东墙补西墙，后者是用杠杆放大收益。

## 模块四：现代场景映射

### 场景一：创业公司的资源分配

你是一家B轮公司的CEO，账上还有12个月现金，市场环境恶劣。原文教你怎么做：

1. **先分"土地"：** 把核心业务线拆出来，给负责人真正的决策权和利润分成。不要用KPI压人，用利益绑人。
2. **砍掉非"农业"项目：** 所有不能直接造血的项目全部暂停。营销活动砍到只剩ROI最高的渠道。团队缩减到核心精锐。
3. **开放"私人经济"：** 把非核心业务外包，让外部供应商来分担成本。不要什么都要自己干——在资源受限期，"可控"比"自营"更重要。
4. **不靠"加税"续命：** 不要通过克扣供应商账期、压低员工薪资来省现金。把精力放在做大核心业务上。

### 场景二：大公司内部的创新业务孵化

你是一家大公司的创新业务负责人，预算被总部卡得死死的。

1. **劳动互助社逻辑：** 人手不够怎么办？不是等总部批HC，是把现有人员的空闲时间组织起来做互助调配。原文的"劳动互助社和耕田队"翻译过来就是**内部人才池和弹性调配机制**。
2. **犁牛合作社逻辑：** 关键生产资料（工具、算力、数据）不够怎么办？集中采购、共享使用，成立"共享资源池"。不要每个项目都去申请独立预算。
3. **妇女参加生产：** 被忽视的劳动力是最廉价的增量。翻译：**跨部门调动被闲置的人力，或者让非核心岗位的人参与关键项目。**

### 场景三：个人副业/自由职业

1. **先自给再出口：** 副业起步阶段，先解决"自己能不能靠这个活着"的问题，不要一开始就想做大平台做品牌。
2. **三种经济模式：** 自营（自己的核心技能）、合作（和其他自由职业者搭伙接大单）、外包（把非核心工作分包出去）。三者配合，逐渐让自营和合作占主导。
3. **节省每一个铜板：** "贪污和浪费是极大的犯罪"——翻译：**在资源受限期，每一分钱的ROI都要算清楚。不买不需要的工具，不做没有直接转化率的投入。**
4. **不靠增税：** 不要想着通过"涨单价"来增加收入，要想办法通过"增加客户数"或"提高复购"来增加收入。前者是抽血，后者是造血。

## 模块五：行动指导SOP检查清单

### 资源受限期经营决策检查清单

#### 第一步：利益绑定检查
- [ ] 核心团队/关键人才是否拥有明确的"产权"？（股权、分成权、决策权）
- [ ] 他们的努力是否能直接转化为个人收益？
- [ ] 是否存在"干多干少一个样"的吃大锅饭现象？
- [ ] 如果明天核心人员离职，损失有多大？为什么留不住？

#### 第二步：优先级检查
- [ ] 当前排名第一的业务是否是"造血型"业务？（不是"融资型"或"情怀型"）
- [ ] 是否有超过30%的资源投入在非核心业务上？
- [ ] "先自给再出口"的逻辑是否被执行？（内部效率是否先于外部扩张）
- [ ] 如果只能保留一个业务线，是哪一个？

#### 第三步：业务结构检查
- [ ] 是否存在"自营/合作/外包"三层结构？还是全自营或全外包？
- [ ] 自营部分是否控制了行业命脉？（核心技能、核心数据、核心客户关系）
- [ ] 合作/外包部分是否有清晰的规则和边界？
- [ ] 三层结构之间是互补关系还是竞争关系？

#### 第四步：财务健康检查
- [ ] 收入增长是否主要来自业务扩张，而非涨价/压榨？
- [ ] 融资节奏是否匹配业务增长？还是为了补窟窿？
- [ ] 是否存在"贪污和浪费"？（无人监管的采购、低ROI的营销投入、冗余的SaaS订阅）
- [ ] 现金流能支撑几个月？如果降到6个月以下，启动了哪些应急措施？

#### 第五步：组织效率检查
- [ ] 关键生产资料（工具/设备/账号）是否实现了共享和最大化利用？
- [ ] 是否有被忽视的"闲置劳动力"可以调配？
- [ ] 跨部门协作是否存在"劳动互助社"式的弹性调配机制？
- [ ] 组织内部是否存在大量"只消耗不产出"的岗位？

---

### 一句话复盘

**所有的经营问题，归根结底是两个问题：蛋糕在不在长大？分蛋糕的人是不是拼命在做蛋糕的人？** 第一个问题决定你能不能活，第二个问题决定你活得累不累。1934年的那篇文章，把这两个问题想透了。
