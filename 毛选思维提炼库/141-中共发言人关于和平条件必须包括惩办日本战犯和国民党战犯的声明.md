# 《中共发言人关于和平条件必须包括惩办日本战犯和国民党战犯的声明》思维提炼报告

---

## 模块一：核心算法一句话总结

**对手每犯一个错，你就多吃一个子——他的错误动作不是麻烦，是给你送的新弹药、新条件、新合法性。**

---

## 模块二：历史困境与破局逻辑

### 当时的绝境

上一轮中共要求国民党重新逮捕冈村宁次、逮捕内战罪犯。国民党政府发言人回应了——但不承认、不执行，反而狡辩：

- "冈村宁次是司法问题，与和谈无关"
- "真正的和平不应该有先决条件"
- 指责中共发言人"不够郑重""节外生枝"

更恶劣的是，国民党不但没有重新逮捕冈村宁次，反而把他和其他260名日本战犯一起送去了日本。

对手在用"话术防御"对抗你的"实质要求"——把你的要求归类为"司法问题""先决条件""节外生枝"，试图在框架上否认你要求的合法性。

### 破局核心操作

中共发言人用了三招连环升级：

**第一招：去合法性——对手已经不配叫"政府"了**
"南京没有行政机关。广州没有行政首脑。上海既没有行政机关又没有行政首脑。奉化只有一个宣布退休的伪总统。"

直接在"政府"二字上加引号——你不是政府，你是一个"假定的或象征的政府"。上次还叫你"政府"是我们的态度"不够郑重"。

**第二招：用对手的行动证明你的新要求合理**
对手说"释放战犯与和谈无关"。发言人反驳：一月十四日提八条时确实无关，但一月二十六日你释放了冈村宁次，这就有关了。一月三十一日你又把260名战犯送到日本，这就更有关了。

**每一个新的事实，都是追加新条件的理由。** 于是在第一条"惩办国内战争罪犯"之外，新增了"（甲）惩办日本战争罪犯"。

**第三招：用讽刺消灭对手的道义立场**
"日本是你们如此热爱的地方，以致使你们相信日本战犯生活在你们统治的区域，还不如使他们生活在日本较为安全些。"

"在头衔上加上'卖国'二字，你们应当承认了。"

## 模块三：顶层思维模型提取

### 模型一：「错误即弹药」递进加码术

**核心逻辑**：在博弈过程中，对手犯的每一个错误都不是"麻烦"——它是你追加新条件的弹药。对手越错，你的条件越多、越硬、越合理。

**递进公式**：
1. 初始条件：惩办国内战争罪犯
2. 对手犯错：释放日本战犯 → 追加条件：惩办日本战争罪犯
3. 对手再犯错：把战犯送去日本 → 再追加：你从"政府"降级为"假定的象征的政府"
4. 对手狡辩："与和谈无关" → 你直接否定他的主体资格

**关键心法**：不要急着一次性把所有条件亮出来。先亮基础条件，等对手犯错后再追加——追加的条件因为"事出有因"，比一开始就提出来更有说服力。

### 模型二：「渐进式去合法性」降级术

**核心逻辑**：不一次性否定对手的合法性，而是根据对手的行动逐步降级——从"政府"到"反动政府"到"卖国政府"到"假定的象征的政府"到加引号的"政府"。

**降级阶梯**：
- 第一级：政府（正常称呼）
- 第二级：反动政府（定性）
- 第三级：反动卖国政府（加重定性）
- 第四级：假定的或象征的政府（否定实体性）
- 第五级："政府"（加引号，彻底否定）

**迁移要领**：每降一级都有对应的事实依据。降级不是情绪宣泄，而是对对手行为的精确回应。你的降级越有事实支撑，对手越无法反击。

### 模型三：「反话正说」讽刺包围术

**核心逻辑**：用极度"礼貌"的语言包裹极度尖锐的内容——表面上是"客气话"，实际上是让对方无地自容。

**示例**：
- "你们是缩短战争时间、减轻人民痛苦、以拯救人民为前提的大慈大悲救苦救难的人们" → 实际：你们在屠杀人民
- "你们有很多的好心眼儿" → 实际：你们心肠黑透了
- "请原谅，我们对你们只能取这种态度" → 实际：你们不配更好的态度

## 模块四：现代场景映射

### 场景一：甲方拖欠尾款后反复找借口

你做完项目，甲方拖欠尾款。你催了，甲方说"财务流程还没走完"，同时你发现甲方在同一个项目上已经找了新的供应商接手。

**用错误即弹药术拆解**：
- 第一轮催款："尾款请尽快结算"（基础条件）
- 甲方借口"流程问题" → 你追加："请提供具体流程节点和时间表"
- 你发现甲方找了新供应商 → 你追加违约条件："在我方尾款结清前不得将项目成果交由第三方使用"
- 甲方继续拖延 → 你升级：从"合作方"降级为"违约方"，发律师函

每一步都有甲方自己的行为作为你追加条件的依据。

### 场景二：投资方尽调后反悔压价

投资方做完尽调后忽然说"估值太高，降50%吧"，同时你发现他们已经用尽调获取的信息去接触了你的竞争对手。

**用渐进式去合法性术拆解**：
- 第一级："投资人"（正常）
- 尽调后压价 → 第二级："试图利用信息不对称的投资方"
- 用尽调信息接触竞品 → 第三级："违反保密协议的商业间谍行为主体"
- 每降一级都有具体事实支撑，最后直接发律师函而不是继续谈估值

## 模块五：行动指导 SOP（实战检查清单）

### 立刻可以问自己的5个反思问题：

1. **对手最近的错误动作，你有没有把它转化为追加条件的弹药？** 每一个错误都值得一个新条件。

2. **你是急着一次性亮底牌，还是留了余地等对手犯错后再追加？** 先亮基础条件，等对手犯错——追加比首提更有力。

3. **对手的"合法性"你有没有根据他的行为逐步降级？** 列出对手的行为，看有没有足够的理由降低他在你心中的信用等级。

4. **你在反驳对手时，有没有把对手的话反过来用？** 对手说"大慈大悲"你就用"大慈大悲"来反讽——用对方的词杀伤力最大。

5. **你的每一项追加条件有没有"事出有因"的事实依据？** 没有事实依据的追加是耍赖，有事实依据的追加是天经地义。
